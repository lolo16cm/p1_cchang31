Project Script Descriptions (Python):
1: draw_cc.py (launched via p1a.launch): This script manages two turtles to draw the initials "CC." It kills the default turtle1 and spawns new turtles specifically for the drawing process. 
reference source:
https://github.com/dhruvsh-1729/Name-IT-Turtlebot-

2: draw_name.py: This script uses the default turtle to draw two "C" shapes by utilizing the teleport_absolute service to move. 
reference source:
https://www.youtube.com/watch?v=Qh15Nol5htM&t=29s

3: runner.py (launched via p1b.launch): This is the "Hunter and Runner" task. The runner moves constantly, while the hunter chases it. If the hunter catches the runner, the current runner is killed and a new one is spawned at a random location, repeating the process. 
reference source:
https://github.com/ctran20/ROS_TurtleSim_Hunt/tree/main/scripts
https://www.youtube.com/watch?v=Qh15Nol5htM&t=29s
